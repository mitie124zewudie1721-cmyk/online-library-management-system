Online Library Management System
Full MERN stack application for managing books, users, borrowing, and overdue records.
Features
- User registration & login (JWT)
- Role-based access: Admin, Librarian, Member
- Borrow/return books
- Overdue management & fines
- Admin dashboard
Folder Structurebackend/     → API & server
frontend/    → React client
How to run locally

# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm start

## Tech Stack
- Backend: Node.js, Express, MongoDB, Mongoose, JWT
- Frontend: React, React Router, Axios, Tailwind/Bootstrap
